# Personal Portfolio Website

Hey there! Welcome to my portfolio project. This is my first attempt at building a personal portfolio website from scratch, and I'm really excited to share it with you.

## What This Project Is About

This is a simple yet beautiful portfolio website that I built to showcase who I am and what I'm learning. It's got everything you'd expect from a personal portfolio - an about section, skills, and a way for people to get in touch.

## What You'll Find Here

- **Home** - A friendly welcome section with a quick intro about me
- **About** - A bit more about who I am and what I do
- **Skills** - The technologies I'm currently learning and working with
- **Contact** - A simple form to reach out to me

## Technologies I Used

- **HTML5** - For the structure and content
- **CSS3** - For all the styling and visuals

I also used some cool external resources to make things look nicer:
- Google Fonts (Inter and Outfit) for nice typography
- Boxicons for the icons throughout the site

## Design Features

I tried to make this look modern and professional while keeping it simple. Here's what I went for:

- Dark theme with green accents (I think it looks pretty sleek)
- Glowing background effects for some visual interest
- Smooth hover animations on buttons and skill cards
- Fully responsive - works on phones, tablets, and desktops
- Clean, readable typography

## How to View It

Just open `index.html` in any web browser and you'll see the site in action. No complicated setup or servers needed - it should work right out of the box.

## What I Learned

While building this, I got hands-on experience with:
- Structuring a webpage with proper HTML
- Styling with CSS (flexbox, grid, and more)
- Making layouts that work on different screen sizes
- Adding some nice visual effects
